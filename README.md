# Edge Duplicate Tab Remover

A simple browser extension (Edge/Chrome) to detect and remove duplicate tabs by URL/title heuristics. This repo contains the popup UI, background worker logic, clustering worker, and session management.

## Features

- Detect duplicate tabs (exact or fuzzy matching)
- Remove duplicates with confirmation modal
- Save and restore named sessions
- Compact UI for toolbar popups and an expandable panel window

## Development

1. Load unpacked extension in Edge/Chrome using this folder.
2. Reload extension after code changes.
